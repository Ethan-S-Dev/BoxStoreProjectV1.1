# Box Store Project WPF
### A Data Structure Class Project

This is a project I did for my data structure class.
Main objective was to make the fastest and must efficant adding, searching and buying boxes.

- **MVVM design pattern**

### Data Structures:

- **Queue**
- **Binary Search Tree**

### Application
- **WPF**
- **EF 6**
- **.NET Framework 4.8**
